fn try_unwrap(rc: Rc<T>) -> Result<T, Rc<T>>

Unwraps the contained value if the Rc<T> is unique.

If the Rc<T> is not unique, an Err is returned with the same Rc<T>.

Examples

fn drop(&mut self)

Drops the Rc<T>.
